Overview Theme aware

This layout uses CSS variables and a data-theme attribute on :root to smoothly switch between light and dark themes.

How it works CSS vars

All colours come from variables like --bg and --text. The dark theme simply overrides those values.

Where to use Flexible

Drop this into a standalone page, or adapt parts of it for dashboards, intranet tiles, or email-style layouts.

Customisation Easy

Update font sizes, swap colours, or plug in your own content blocks while keeping the same theme system.
